# FlyMilhas - Planejamento do Template

## Visão Geral
Template HTML/CSS para SAAS de gestão de milhas aéreas, focado em simplicidade e funcionalidade para integração com n8n.

## Páginas Necessárias

### 1. Página de Login (login.html)
- Formulário de login
- Link para registro
- Opção "Esqueci minha senha"
- Design clean e profissional

### 2. Dashboard Principal (index.html)
- Visão geral das milhas
- Gráficos de saldo e histórico
- Ações rápidas
- Notificações importantes

### 3. Gestão de Milhas (milhas.html)
- Lista de contas de milhas
- Adicionar/editar contas
- Visualizar saldos
- Histórico de movimentações

### 4. Transações (transacoes.html)
- Histórico completo
- Filtros por data, tipo, companhia
- Detalhes das transações
- Exportar dados

### 5. Perfil/Configurações (perfil.html)
- Dados pessoais
- Configurações de notificação
- Preferências do sistema
- Gerenciar conta

- Botões (primário, secundário, outline)
- Cards informativos
